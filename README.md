# Задание к лекции «Потоки ввода-вывода. Работа с файлами. Сериализация»

 ## Задание 1
  #### Функции:
  * initFiles() - инициализация переменных.

 ## Задание 2

  * saveGame() - сохранение игры в файлах.

 ## Задание 3
  #### Функции:
  * openProgress() - загрузка игры из файлов.
